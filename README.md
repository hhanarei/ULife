# ULife

ULife is a web-site designed to centralize academic and personal life management for university students. It is categorized within the fields of Education Technology (EdTech) and Enterprise Web Applications.

## Problem
Students currently jump between Moodle, emails, and budgeting apps. This leads to missed deadlines and high stress. ULife centralizes academic tracking, financial planning, and wellbeing monitoring into one intelligent dashboard.

## Features
- Academic Dashboard: Provides a unified view of schedules, deadlines, and a GPA tracker.
- Smart Analytics: Visualizes data regarding GPA trends and workload.
- Budget & Expense Tracker: Tools for managing personal financial planning.
- Wellbeing Check-in: A module dedicated to monitoring student mental health.
